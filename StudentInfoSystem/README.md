3. Student Info System with GUI  

Goal:  
Create a GUI-based app to manage student records such as name, roll number, department, marks, and grades.  
Use JavaFX or Swing for the user interface.

Resources:  
• JavaFX Tutorial  
• Swing vs JavaFX Comparison  
• Simple GUI App - Java Swing  

Skills Covered:  
• JavaFX or Swing (GUI framework)  
• Event handling and layout design  
• GUI logic and validation  
• TableView or JTable usage  

Make It Your Own:  
• Add a grading calculator that auto-calculates grades based on marks.  
• Display all student records in a TableView (JavaFX) or JTable (Swing).  
• Allow add, update, delete, and search functionalities.  
• Validate duplicate roll numbers or invalid input.  
• Save and load data from CSV or text file (optional).
