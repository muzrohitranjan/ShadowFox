# Enhanced Console Calculator

A comprehensive Java console calculator with basic arithmetic, scientific calculations, and unit conversions.

## Features

### 1. Basic Arithmetic
- Addition, Subtraction, Multiplication, Division
- Error handling for division by zero

### 2. Scientific Calculations
- Square Root
- Power (Exponentiation)
- Trigonometric functions (Sine, Cosine)
- Logarithm (base 10)

### 3. Unit Conversions
- **Temperature**: Celsius ↔ Fahrenheit, Celsius → Kelvin
- **Length**: Meters ↔ Feet, Kilometers → Miles
- **Weight**: Kilograms ↔ Pounds, Grams → Ounces

## How to Run

1. Compile: `javac EnhancedCalculator.java`
2. Run: `java EnhancedCalculator`

## Error Handling
- Invalid input validation
- Mathematical error checks (negative square roots, division by zero)
- Exception handling for input mismatches
